# 文档主标题（用 H1，写法清晰）

## 一级章节（用 H2，每个 H2 是一段独立内容）

正文段落。RAG 会按 H2 / H3 边界切片，所以每段 H2 应该聚焦一个独立的小主题。

如果是步骤 / 列表 / 表格，可以用 markdown 语法：

1. 第一步
2. 第二步
3. 第三步

或者表格：

| 字段 | 取值 | 说明 |
| --- | --- | --- |
| type | string | 数据类型 |
| required | boolean | 是否必填 |

## 另一个 H2 章节

如果文档很长，每个 H2 内部还可以用 H3：

### 子章节

正文...

### 另一个子章节

正文...
